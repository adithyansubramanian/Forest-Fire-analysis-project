# Forest-Fire-analysis-project
A simple project analyzing Portuguese forest fire data 

# Getting Started
To view the code you will need Rstudio. 

# Data Source
The data can be found in the repository as forestfires.csv, or can be found at https://archive.ics.uci.edu/ml/machine-learning-databases/forest-fires/. The data is from Paulo Cortez and AnÄ±bal Morais' paper, A Data Mining Approach to Predict Forest Fires using Meteorological Data.
